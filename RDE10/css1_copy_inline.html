CSS: Cascading Style Sheets

Objetiva realizar a estilização do conteúdo de documentos HTML. Podem definir cores, fontes, espaçamentos, sombras, entre outros.

Sintaxe

Formas de utilizar o CSS

Inline

  • Os códigos são definidos dentro de uma tag HTML.
  • Afeta somente a tag em que foi definida.
    •

    Incorporado

  • Os códigos são definidos no cabeçalho, dentro da tag head.
  • Afeta somente o documento em que foi definido.
    •

    Externo

  • Os códigos CSS são definidos em um arquivo externo.
  • Afeta todos os documentos que estão linkados ao CSS.
