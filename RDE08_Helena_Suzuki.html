<table>

Possibitila a organização de dados em linhas, colunas e células, ou seja, essa tag serve para criar uma tabela no HTML.

Tags complementares

Além da tag <table>, é necessária a utilização de algumas outras para criar uma tabela completa.

<tr>: Determina o início e o fim de uma linha da tabela.

<td>: Define o conteúdo que estará dentro de cada célula.

<th>: Define o cabeçalho da tabela.

Atributos

Os atributos de tabelas geralmente são empregados no CSS, mas serão apresentado alguns básicos que podem ser utilizados no próprio HTML.

border: Define a largura da borda.

cellspacing: Define a distância entre as células da tabela.

cellpadding: Define a distância entre a “borda” da célula até o seu conteúdo.

Exemplo

Código implementado:


Nome Escola
Carlos Ribeiro Xavier
Bianca Dias IFPR

<del>

Representa uma parte do texto que foi excluída de um documento. Esse fragmento é, geralmente, renderizado pelos navegadores com uma linha "cortando" o texto.

Exemplo

Código implementado:

This text has been deleted

<ins>

É utilizado para quando um texto foi inserido ao documento.

Exemplo:

Código implementado:

Esse texto foi adicionado.

<sub>

Essa tag determina que o texto que esteja embutido nela fique subscrito ao restante. Nesse sentido, ele usualmente ficará numa linha de base um pouco mais baixa e com uma fonte menor. Ela pode ser utilizada para notas de rodapé, fórmulas químicas e subscritos de variável.

Exemplo

Código implementado:

A fórmula molecular da glicose é C6H12O6.

<sup>

Ao utilizar essa tag, o texto embutido nela ficará sobrescrito ao restante. Sendo assim, ele geralmente aparecerá em uma linha de base um pouco acima e com uma fonte menor. Essa tag é comemunte utilizada para números expoentes, mas também poder ser usada para letras.

Exemplo

Código implementado:

22=4

<blockquote>

A tag indica que o texto incluído é uma longa citação. Usualmente, é apresentado como um recuo no texto. A URL para a fonte da citação pode ser dada usando o atributo cite, enquanto uma representação de texto da fonte pode ser dada usando a tag <cite> elemento.

Exemplo

Código implementado:

Esta é uma citação tirada da Mozilla Developer Center.

<q>

O elemento HTML <q> indica que o texto dentro da tag é uma pequena citação. Este elemento destina-se a citações curtas que não requerem marcações de parágrafo.

Atributos

Além de incluir os atributos globais, ele possui o cite, que tem como valor um URL, o qual encaminhará para a fonte da informação citada.

Exemplo

Para mais informações clique aqui.

<abbr>

Essa tag irá representar uma abreviação e opcionalmente fornece uma descrição completa para ela. Se presente, o atributo title deve conter a descrição completa.

Exemplo

Código implementado:

Obama é presidente dos EUA.

<address>

Ela fornece informações de contato para seu ancestral <article> ou <body> mais próximo; no segundo caso, ele se aplica ao documento inteiro.

Exemplo

O seguinte conteúdo está expresso entre as tags "address":

Voce pode contatar o autor em www.somedomain.com.
Se encontrar qualquer bug, por favor contate o administrador do site. Você tambem pode querer nos visitar:
Mozilla Foundation
1981 Landings Drive
Building K
Mountain View, CA 94043-0801
USA

Além de sua função semântica, ele deixa o texto como se esse estivesse entre <i>.

<cite>

O elemento HTML <cite> representa uma referência a um trabalho artístico. Deve incluir o título do trabalho ou uma URL de referência, que pode ser em uma forma abreviada de acordo com as convenções usadas para a adição dos metadados de citação.

Exemplo

Código implementado:

Mais informações podem ser encontradas em [ISO-0000].

<bdo>

O elemento HTML <bdo> (bidirectional override) é utilizado para substituir a direcionalidade atual do texto. Isso fará com que a direção do texto se torne aquela que foi especificada no código.

Atributos

dir indicará a direção do texto. Os seus valores possíveis são:

ltr: com esse, o texto ficará da esquerda para a direita (Left-To-Right), como o usual.
rtl: o texto ficará da direita para a esquerda (Right-To-Left).

Exemplo

Código implementado:

Este texto ficará da esquerda para a direita.

Este texto ficará da direita para a esquerda.

<code>

Essa tag define um fragmento de código de computador no documento HTML. Os navegadores tradicionalmente renderizam o texto encontrado na tag <code> em fonte monoespaçada.

Exemplo

Código implementado:

Computer code goes here

<pre>

HTML texto preformatado <pre> é a tag utilizada para representar um texto pré-formatado. O texto dentro desse elemento é exibido da mesma maneira em que o texto original. Ele geralmente fica desproporcional ao restante do conteúdo e os espaços em branco são mantidos no texto, assim como ele foi digitado.

Exemplo

<kbd>

O elemento HTML representa uma extensão de texto embutido denotando a entrada textual do usuário a partir de um teclado, entrada de voz ou qualquer outro dispositivo de entrada de texto. Por convenção, o padrão do agente do usuário é renderizar o conteúdo de um elemento usando sua fonte monoespaçada padrão, embora isso não seja obrigatório, pois pode ser estilizado com CSS.

Exemplo

Código implementado:

Para copiar algum elemento, digite ctrl + c, e para colá-lo, ctrl + v.
