Software de Versionamento

Sistema que tem por objetivo realizar o controle de versões de um código ou documento.

É possível gerenciar as versões de uma código ou documento realizadas por um desenvolvedor ou equipe de desenvolvedores.

Tipos de Software de Controle de Versão

Git

É um software livre de versionamento que surgiu para o desenvolvimento do Kernel do Linux, foi projetado e desenvolvido por Linus Torvalds em 2005.

Cada diretório de trabalho do Git é um repositório com um histórico ocmpleto e habilidade total de acompanhamento das revisões, não dependente de acesso a uma rede ou a um servidor central.

GitHub

Plataforma de hospedagem de código-fonte e arquivos com controle de versão da Microsoft.

Softwares que serão utilizados:

  • GitHub
  • Git
  • GitHub Desktop
  • Visual Studio Code

Comandos GIT

commit: Torna permanente um conjunto de alterações em um repositório.

push: Envia alterações para o servidor no repositório remoto.

fetch: Verifica de o seu repositório local está em sincronia com o remoto.

pull: Sincroniza o repositório local com o remoto atualizando as versões.
